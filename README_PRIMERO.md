# 👋 ¡BIENVENIDO A RUIZ!

## ¿QUÉ ES RUIZ?

**Ruiz** es tu asistente de voz inteligente. Controla tu celular completamente por voz. Solo necesitas decir **"Ey Ruiz"** y lo que quieres hacer.

---

## 🚀 COMENZAR EN 30 SEGUNDOS

### Paso 1: Abre la Aplicación
```
Abre el archivo: APP.php
(Doble clic o arrastra a tu navegador)
```

### Paso 2: Permite el Micrófono
```
Haz clic en "Permitir"
```

### Paso 3: Toca el Botón 🎤
```
Presiona el botón circular en el centro
```

### Paso 4: Di "Ey Ruiz"
```
Ruiz te responderá: "¿Qué necesitas?"
```

### Paso 5: Pide algo
```
Ej: "¿Qué hora es?"
```

**¡Listo! Ya estás usando Ruiz.**

## 🆘 SI ALGO NO FUNCIONA

1. **Ruiz no me escucha**
   → Habla más claro, busca un lugar tranquilo
   → Intenta "Hey Ruiz" o "Ei Ruiz"

2. **No escucho a Ruiz**
   → Sube el volumen del dispositivo
   → Intenta con auriculares

3. **APP.php no se abre**
   → Doble clic en APP.php
   → O arrastralo a tu navegador
   → O abre manualmente en el navegador
