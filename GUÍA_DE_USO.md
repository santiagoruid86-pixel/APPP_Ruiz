## 🆕 Primer Inicio

### Paso 1: Preparación
1. Asegúrate de tener un navegador moderno (Chrome, Edge, Firefox o Safari)
2. Verifica que tu dispositivo tenga micrófono
3. Comprueba que el volumen esté activado
4. Encuentra un lugar tranquilo para mejor reconocimiento

### Paso 2: Abrir la Aplicación
- **En Computadora**: Abre `APP.php` en tu navegador
- **En Celular**: Abre `APP.php` en Chrome o Safari
- **En Tablet**: Abre `APP.php` en el navegador

### Paso 3: Permitir Permisos
Cuando veas el popup que pide permisos:
- ✅ Haz clic en "**Permitir**" para el micrófono
- ✅ Haz clic en "**Permitir**" para el altavoz
- Estos permisos solo se usan localmente en tu dispositivo

### Paso 4: Primera Prueba
1. Toca el botón circular 🎤 en el centro
2. Espera a que el botón se ponga verde (está escuchando)
3. Di claramente: **"Ey Ruiz"**
4. Ruiz debería responder: "¿Qué necesitas?"

## 🎤 Cómo Hablar con Ruiz

### Formato de Comando
```
[Activación] + [Comando]
```

### Ejemplo 1: Preguntar la Hora
```
"Ey Ruiz, ¿qué hora es?"
```
Ruiz responderá con la hora actual.

### Ejemplo 2: Contar un Chiste
```
"Ey Ruiz, cuéntame un chiste"
```
Ruiz contará un chiste divertido.

### Ejemplo 3: Abrir una Aplicación
```
"Ey Ruiz, abre WhatsApp"
```
Se abrirá WhatsApp Web en una nueva pestaña.

### Consejos para Mejor Reconocimiento
- 🗣️ Habla claro y a velocidad normal
- 🤐 No hables demasiado rápido ni muy lento
- 🔇 Evita ambientes ruidosos
- 📱 Acerca el micrófono a tu boca (no demasiado)
- ⏸️ Haz una pausa después de cada frase

## 🆘 Solución de Problemas

### ❌ Problema: Ruiz no me escucha
**Posibles causas y soluciones:**

1. **Micrófono no permitido**
   - Ve a Configuración del navegador
   - Busca "Permisos de sitio"
   - Permite acceso al micrófono

2. **Micrófono desactivado en el sistema**
   - Abre Configuración de Sonido
   - Verifica que el micrófono esté activo

3. **Ruido de fondo excesivo**
   - Busca un lugar más tranquilo
   - Cierra otras aplicaciones que usen micrófono

4. **Micrófono defectuoso**
   - Prueba con auriculares con micrófono
   - Testa el micrófono en otra aplicación

---

### ❌ Problema: Ruiz no me habla
**Posibles causas y soluciones:**

1. **Volumen silenciado**
   - Verifica el volumen del dispositivo
   - Sube el volumen en el navegador
   - Testa con auriculares

2. **Altavoz dañado**
   - Prueba con auriculares
   - Intenta en otro dispositivo

3. **Síntesis de voz desactivada**
   - Verifica permisos de audio
   - Recarga la página
   - Intenta en otro navegador

---

### ❌ Problema: "Ey Ruiz" no es reconocido
**Posibles causas y soluciones:**

1. **Pronunciación incorrecta**
   - Di "EY" (como en "Hey")
   - "RWIZ" (no "ROOS")
   - Con acento hispanohablante

2. **Acento regional**
   - Intenta con "Ei Ruiz" (sin la 'y')
   - Prueba "Hey Ruiz" (pronunciación inglesa)
   - Intenta "Oye Ruiz"

3. **Ruido de fondo**
   - Busca un lugar más silencioso
   - Habla más lentamente

4. **Idioma del navegador**
   - Verifica que el idioma sea es-ES
   - Abre Developer Tools y revisa console.log()

---

### ❌ Problema: La aplicación va lenta
**Soluciones:**

1. **Limpia caché del navegador**
   - Ctrl+Shift+Delete (o Cmd+Shift+Delete)
   - Selecciona "Caché"
   - Haz clic en "Limpiar"

2. **Cierra otras pestañas**
   - Libera memoria del navegador

3. **Reinicia el navegador**
   - Cierra completamente
   - Reabre la aplicación

4. **Intenta otro navegador**
   - Prueba Chrome si usas Firefox
   - Intenta Safari en iOS

---

### ❌ Problema: Los comandos no funcionan
**Soluciones:**

1. **Espera a que Ruiz responda**
   - Después de decir "Ey Ruiz", espera su respuesta
   - Luego da el comando

2. **Pronuncia el comando correctamente**
   - Di el comando completo
   - Evita abreviaturas

3. **Revisa la pantalla**
   - Mira en "Texto reconocido" lo que entendió
   - Si está mal, prueba de nuevo

## 💡 Consejos y Trucos

### Optimizar Reconocimiento de Voz

1. **Ambiente de Escucha**
   - Usa en ambientes tranquilos
   - Evita cafeterías o lugares públicos ruidosos
   - Cierra ventanas si hay tráfico afuera

2. **Posición del Micrófono**
   - Acerca el micrófono a tu boca (5-10 cm)
   - No respires directamente al micrófono
   - No cierres la boca completamente

3. **Velocidad de Habla**
   - Habla a velocidad natural
   - No demasiado rápido (Ruiz no entiende)
   - No demasiado lento (parece robótico)

4. **Claridad de Pronunciación**
   - Articula bien las palabras
   - Separa las sílabas claramente
   - No arrastres palabras

### Usar el Historial

El historial muestra todos tus comandos:
- 📜 Mira la sección "Historial" al final
- 🔙 Usa comandos anteriores como referencia
- ⏰ Cada comando tiene hora exacta

### Personalizar la Aplicación

Edita `config.js` para:
- Cambiar idioma (es-ES, en-US, etc.)
- Ajustar velocidad de síntesis
- Cambiar tono de voz
- Agregar mensajes personalizados

**Ejemplo:**
```javascript
// En config.js
RUIZ_CONFIG.language = 'es-MX'; // Cambiar a mexicano
RUIZ_CONFIG.speechRate = 0.8;   // Más lento
RUIZ_CONFIG.speechPitch = 1.2;  // Más agudo
```

```

```

**Ejemplos:**

## 📞 Soporte Técnico

Si tienes problemas:

1. **Verifica requisitos**
   - Navegador moderno (Chrome 25+, Firefox 25+, Safari 14.1+, Edge 79+)
   - Conexión a internet
   - Micrófono y altavoz funcionales

2. **Recarga la página**
   - Presiona F5 o Ctrl+R
   - Cierra y reabre el navegador

3. **Limpia datos**
   - Limpia caché del navegador
   - Borra cookies del sitio
   - Intenta en navegación privada/incógnito

4. **Intenta otro navegador**
   - Si funciona en otro, es problema del navegador
   - Si no funciona en ninguno, revisa el micrófono
